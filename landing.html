在首批使用中，您会爱上的三个设计

本地 Skill 库挂载

通过软链接方式将统一开发的 Claude Code skills 挂载到项目目录下，在保持 Skill 代码单一来源的同时，完美实现跨项目的代码复用。

n8n 式流程拖拽

专为非线性自动化场景设计，连线编辑器拥有流畅的连线机制。直观管理 Agent 节点的前置条件与后续分流，编排无需书写胶水代码。

SQLite 运行日志

应用底层引入轻量且高效的 SQLite 作为审计日志数据库。精确记录每一次定时执行的时间、执行路径、各节点的输入输出参数，方便随时回档排查。

底层架构设计

让 Agent 执行稳如磐石

通过在 Tauri 底层引入 Rust 子进程管理和本地文件系统操作，实现对外部 AI 工具的稳定监控与调度。

统一的 Agent 适配层

适配 Claude Code Agent SDK 以及 Codex OpenAI app-server 协议，随时可以按需切换底层大模型。
