content-writer

Generates markdown drafts from outlines using Claude Code.

Usage

Call this skill with an outline, optional tone and length. It returns a markdown string.

{
  "outline": "Your article outline here",
  "tone": "professional",
  "length": 1000
}

Requirements

Requires Claude Code access. No additional dependencies.
